Claude Code

Anthropic 官方推出的 AI 编程助手。运行依赖 Node.js (v18+)。

环境准备

确保您的本地环境已安装 Node.js 及其管理工具。我们建议使用 Homebrew 安装。

bash
brew install node
bash
node --version # v18.0.0+
npm --version # v9.0.0+

安装 Claude Code

macOS / Linux

npm install -g @anthropic-ai/claude-code

Windows (PowerShell)

npm install -g @anthropic-ai/claude-code

连接服务

info
提示： Pigcoder 完全兼容 Claude Code 原生协议。您只需在配置文件中修改 `endpoint` 即可无缝切换。
操作系统 配置路径
macOS / Linux ~/.claude/settings.json
Windows %USERPROFILE%\.claude\settings.json
settings.json data_object
{
  "endpoint": "https://api.pigcoder.com",
  "apiKey": "YOUR_PIGCODER_API_KEY"
}
⚠️

端点地址不要加 /v1，Claude Code 使用 Anthropic 原生协议，直接填入域名即可。

验证配置

运行以下命令确认安装成功并能够连接到服务：

claude --version
如果返回版本号，说明您已准备好开始使用 AI 赋能的编程之旅。
